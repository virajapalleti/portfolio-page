## Overview

City-wide AQI averages hide the fact that pollution in Delhi varies sharply from one neighbourhood to the next. This project pulls hyperlocal sensor data from OpenAQ and weather features from Open-Meteo, then trains an XGBoost model to forecast AQI at the station level.

The frontend is a Plotly Dash dashboard so results are readable rather than trapped in a notebook.

## Tech Stack

Python, XGBoost, pandas, Plotly Dash, OpenAQ API, Open-Meteo API
