## Overview

Classic MNIST digit classification, tackled with two very different approaches. Logistic regression as a linear baseline, K-nearest neighbours as an instance-based comparison. Both are trained on the raw pixel features with light preprocessing, evaluated against each other on the Kaggle test set.

A small exercise in seeing where model choice actually matters versus where the data does most of the work.

## Tech Stack

Python, scikit-learn, NumPy, pandas
