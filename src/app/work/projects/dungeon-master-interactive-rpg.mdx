## Overview

An interactive RPG where the game master is an LLM, but the game logic isn't. Dungeon Master uses LangGraph to hold persistent game state across turns — inventory, character stats, world facts — instead of leaning on the model to remember everything.

Game mechanics (rolls, combat, checks) are surfaced as tool calls the model invokes rather than narrates, and human-in-the-loop interrupts pause the graph at decision points so the player actually plays instead of just watching.

## Tech Stack

Python, LangGraph, LLM Prompt Engineering
