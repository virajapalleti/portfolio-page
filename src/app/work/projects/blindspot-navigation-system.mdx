## Overview

BlindSpot is a wearable navigation system that runs object detection on-device rather than offloading to a phone or server. YOLOv8 is deployed on a Raspberry Pi through NCNN, which handles the ARM-side optimisation so inference stays real-time on limited hardware.

Detections drive directional haptic feedback through a companion mobile app, translating spatial information into vibration patterns the user can feel and act on.

Won 1st place among 40+ teams at the hackathon it was built for.

## Tech Stack

Python, YOLOv8, NCNN, Raspberry Pi, Flask, Svelte, OpenCV
