## Overview

Refract ingests PDFs and adapts its explanations to match how the user actually writes. It builds a small stylistic profile from user-provided text samples — sentence rhythm, lexical choices, register — and uses transformer-based semantic similarity to shape its output.

Style transfer is grounded in linguistic features rather than vibes, and evaluated against a proper eval framework so improvements aren't just qualitative.

## Tech Stack

Python, HuggingFace Transformers, Sentence Transformers, PyPDF2, pyttsx3
